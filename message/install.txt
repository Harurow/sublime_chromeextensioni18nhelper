ChromeExtensionI18nHelper
=========================

Thank you for install this plugin.
please setup this plugin settings.

1. Run “[Preferences] - [Package Settings] - [ChromeExtensionI18nHelper] - [Settings -User]”

2. input current lang. (default 'en')
{
	"default_locale": **"INPUT DEFAULT LOCALE"**
}

Valid location is the following.
check => 'https://developers.google.com/chrome/web-store/docs/i18n?hl=ja&csw=1#localeTable'
